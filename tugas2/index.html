    Metode Newton Raphson¶

    Metode Newton Rapshon merupakan metode pendekatan yang menggunakan satu titik awal x_0 dan mendekatinya dengan memperhatikan gradien pada titik tersebut. Metode ini dimulai dengan mencari garis singgung kurva pada titik (x_i,f(x_i))

    Perpotongan garis singgung dengan sumbu x yaitu X_{i+1}, akan menjadi nilai x yang baru, dengan cara dilakukan berulang-ulang (iterasi). $$ X_{n+1} = x_{n} + \frac{F(x_{n})}{F^{1}(x_{n})} $$

    Persamaan Non Linear¶

    Persamaan non-linier dapat diartikan sebagai persamaan yang tidak mengandung syarat seperti persamaan linier, sehingga persamaan non-linier dapat merupakan:

    1. Persamaan yang memiliki pangkat selain satu (misal: x_2)
    2. Persamaan yang mempunyai produk dua variabel (misal: xy)

    Algoritma Metode Newton Raphson¶

    Algoritma Metode Newton Raphson adalah sebagai berikut:

    1. Definisikan fungsi f(x) yang akan dicari akarnya.

    2. Tentukan harga awal / titik awal (x_0).

    3. Tentukan toleransi kesalahan (ɛ).

    4. Cari turunan fungsi f(x). Jika f ’(x) = 0, maka metode newton raphson tidak dapat dilanjutkan.

    5. Hitung nilai fungsif(x) dan f ’(x) dengan menggunakan titik awal.

    6. Hitung nilai x_{i+1} menggunakan rumus: $$ X_{n+1} = x_{n} + \frac{F(x_{n})}{F^{1}(x_{n})} $$

    7. Hitung kesalahan \left | x_b - x_0 \right | dan bandingkan dengan toleransi kesalahan (ɛ).

    8. Jika \left | x_b - x_0 \right | \leq \varepsilon, maka dipilih akar persamaan x_{i+1} . Jika \left | x_b - x_0 \right |> \varepsilon , maka iterasi dilanjutkan.
    9. Akar persamaannya adalah x_{i+1} yang terakhir diperoleh.

    Implementasi¶

    Tentukan akar dari f(x)=x^6-x-1. Pada titik awal x_0 = 1,5 dan \varepsilon = 0.0001.

    Penyelesaian :

    • f(x)=x^6-x-1

    • Titik awal x_0 = 1,5

    • Toleransi kesalahan \varepsilon = 0.0001

    • Turunan fungsi f(x) adalah f'(x)=6x^5-1. Karena f'(x) \neq0 maka metode newton raphson dapat dilanjutkan.

    • Nilai fungsi f(x) dan f'(x) adalah :

    f(1,5)=(1,5)^6-1,5-1=8,890

    f'(1,5)=6(1,5)^5-1=44,562

    • Nilai x_{i+1} adalah:

    x_{i+1}=x_i-\frac{f(x_i)}{f'(x_i)}

    x_{0+1}=x_0-\frac{f(x_0)}{f'(x_0)}

    x_1=1,5-\frac{8,890}{44,562}=1,3005

    • Kesalahan \left | x_{i+1}-x_i \right | = \left | 1,3005-1,5 \right |= 0,1995

    Iterasi selanjutnya dilakukan dengan mencari f(x_1), f'(x_1),x_{i+1} dan seterusnya sampai \left | x_{i+1}-x_i \right |, sehingga didapatkan data pada tabel berikut:

    n x_1 f(x_i) f'(x_1) x_{i+1} \left\| x_i+1-x_i\right\|

    1 1,300491 2,537266625 21,31968214 1,181480472 0,119010528

    Sampai iterasi ke-4, didapat \left| x_{i+1} - x_i\right| \leq \varepsilon yaitu 0,000053546 \geq 0,0001, maka iterasi dihentikan dan didapat nilai akar x yaitu x_{i+1} = 1,134724145.

    Berikut program dari contoh soal di atas:

    import math
e = 2.71828

def fungsi(x):
    x = float((e**x) - (4*x))
    return x

def fungsiturunan(x):
    x = float((e**x) - (4))
    return x

x = float(input('Masukkan nilai awal = '))
error = float(input('Masukkan nilai error = '))
selisih = 1
if fungsi(x) == 0:
    print("Tidak dilanjutkan karena fungsi(x) = 0")
else:
    iterasi = 0
    while selisih>error :
        iterasi += 1
        x1 = x - (fungsi(x)/fungsiturunan(x))
        selisih = math.fabs(x1 - x)
        x = x1
        print("Iterasi ke = ",iterasi,", x = ",x1, ", f(",x1,") = ",fungsi(x1),", selisih = ",selisih)

    Penjelasan :

    1. Import Library Math
    2. Contoh fungsi yang digunakan pada program diatas ialah f(x)=x^6-x-1, dan turunan fungsinya ialah f'(x)=6x^5-1 , maka menghasilkan f'(x) \neq 0 sehingga metode newton raphson dapat dilanjutkan.
    3. Membuat sebuah inputan untuk X , Error / Epsilon
    4. Deklarasikan selisih = 1 untuk pengecekan apakah selisih lebih besar dari error
    5. Deklarasikan iterasi = 0 untuk perulangan yang ke 0 nantinya dan akan ditambah setiap kali perulangan apabila pengecekan pada step nomor 4 telah terpenuhi
    6. Deklarasikan selisih untuk x_b - x_0 untuk perbandingan
    7. lakukan perulangan dengan kondisi iterasi kurang dari sama dengan inputan maksimal iterasi dan selisih lebih dari error / epsilon
    8. hitung x_b dengan rumus yang sudah kita dapatkan sebelumnya
    9. lalu lakukan perbandingan jika mencapai nilai True maka toleransi tidak terpenuhi namun perulangan sudah mencapai batas
    10. jika pengecekan selisih > error bernilai True maka toleransi akan terpenuhi dengan nilai error serta fungsi x pada iterasi ke n
